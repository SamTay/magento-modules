BlueAcorn Ajax Mini Cart
========================

This module adds ajax functionality to the product and category pages so that adding a product to the cart is done via ajax and the mini-cart is shown.

Assumptions
----------
- This module can only be used for Enterprise Sites and has only been tested with simple products.
- Deleting products is not done via ajax in mini cart.
- Customizations will need to be made for custom themes.
- The system configuration in sales->checkout->cart->redirect to cart will need to be turned to 'No' in order for the module to be enabled.
